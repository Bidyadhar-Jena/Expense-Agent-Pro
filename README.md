# AI Expense Agent Pro 💰

AI Expense Agent Pro is a smart expense tracking web application built using HTML, CSS, and JavaScript. It helps users manage daily expenses, automatically categorize transactions, and visualize spending insights with charts.

---

## 🚀 Features

- User Registration & Login
- Add Daily Transactions
- Automatic Expense Categorization
- Expense Charts & Visualization
- Responsive User Interface
- LocalStorage & SessionStorage Support

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
- SessionStorage API
- HTML5 Canvas API
